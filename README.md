# NeuralNetwork_From_Scratch
A Fully-connected Neural Network, build from scratch using only Numpy (and Sklean for standardizing the data).

## How to Run
python NN.py / python3 NN.py

### Prerequisites
- Numpy.
- Python/Python3.
- Sklearn.
